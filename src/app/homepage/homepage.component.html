Welcome to the Library Application

This is a training project to demonstrate a simple library application. It showcases CRUD operations for managing books, and it provides features like adding, editing, and deleting books.

Project Features:

  • View a list of books with their details.
  • Add new books to the library.
  • Edit existing book information.
  • Delete books from the library.
  • Responsive design for mobile and desktop.
